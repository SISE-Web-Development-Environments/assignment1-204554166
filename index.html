Difference between Web 1.0, Web 2.0 and Web 3.0:
The first most biggest differences between this 3 versions of the web, the web 1.0 was mostly read-only environment while the web 2.0 was wildly read-write, but the biggest next step was at the web 3.0 version where the environment is portable and personal. The second difference between these versions is their purposes meaning the web 1.0 mostly can hadle only home pages, the web 2.0 contained more blogs and wikis, and now at the web 3.0 version is dealing with live-streams. The third but not last of differences between the web versions is dealing about advertisments, web 1.0 has never had advertising it was banned, the web 2.0 contained only interactive advertising, and the web 3.0 behavioural advertising.
Lets take an example of searching content in different web versions:
web 1.0:
1. The user enter his web browser.
2. The user type the exact web URL into the web address bar.
3. The user search the content he desired in the specific website.
As a result -> only if the user know the website url he will be able to search the content, moreover, the information he need could not be in the website he entered.
web 3.0:
1. the user enter his web browser
2. the user search the information he need in the URL bar
3. the info wil be introduced to hi as a list
4. the user will choose the best url that hold the information.
As a result -> the user can choose which website gives him the best info he needed as he scan the list of URLs that came as a result of his search query.
